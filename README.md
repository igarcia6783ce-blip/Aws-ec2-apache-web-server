# AWS EC2 Apache Web Server Project

## 🚀 Overview
Deployed a live web server using AWS EC2 and Apache.

## 🛠️ What I Did
- Launched an EC2 instance (Amazon Linux)
- Installed Apache (httpd)
- Started and enabled the web server
- Configured security groups (opened port 80 for HTTP)
- Created and deployed a custom HTML page
- Verified public access via browser
